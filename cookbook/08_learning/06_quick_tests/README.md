# 06_quick_tests

Quick validation scripts for critical learning paths.

## Files

- `01_async_user_profile.py`: Async learning path smoke test.
- `02_learning_true_shorthand.py`: `learning=True` shorthand initialization test.
- `03_no_db_graceful.py`: Graceful behavior when learning is enabled without a database.
- `04_claude_model.py`: Learning behavior test with Claude model integration.
