# 04_workflows/07_cel_expressions/loop

## Scope
Runnable workflow examples under: cookbook/04_workflows/07_cel_expressions/loop

## Files
- cel_compound_exit.py: Demonstrates cel compound exit.
- cel_content_keyword.py: Demonstrates cel content keyword.
- cel_iteration_limit.py: Demonstrates cel iteration limit.
- cel_step_outputs_check.py: Demonstrates cel step outputs check.

## Prerequisites
- Activate the demo environment: .venvs/demo/bin/python
- Load API keys with direnv allow (requires a local .envrc file).
- CEL examples require cel-python installed.
