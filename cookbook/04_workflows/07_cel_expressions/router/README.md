# 04_workflows/07_cel_expressions/router

## Scope
Runnable workflow examples under: cookbook/04_workflows/07_cel_expressions/router

## Files
- cel_additional_data_route.py: Demonstrates cel additional data route.
- cel_previous_step_route.py: Demonstrates cel previous step route.
- cel_session_state_route.py: Demonstrates cel session state route.
- cel_ternary.py: Demonstrates cel ternary.
- cel_using_step_choices.py: Demonstrates cel using step choices.

## Prerequisites
- Activate the demo environment: .venvs/demo/bin/python
- Load API keys with direnv allow (requires a local .envrc file).
- CEL examples require cel-python installed.
