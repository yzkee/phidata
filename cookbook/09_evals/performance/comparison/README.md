# Performance Comparison Cookbooks

These benchmarks compare agent instantiation patterns across non-Agno frameworks.

## Files

- `autogen_instantiation.py` - AutoGen assistant instantiation benchmark.
- `crewai_instantiation.py` - CrewAI agent instantiation benchmark.
- `langgraph_instantiation.py` - LangGraph react-agent instantiation benchmark.
- `openai_agents_instantiation.py` - OpenAI Agents SDK instantiation benchmark.
- `pydantic_ai_instantiation.py` - PydanticAI agent instantiation benchmark.
- `smolagents_instantiation.py` - Smolagents tool-calling agent instantiation benchmark.
