# Other Team Patterns

Additional team patterns that don't fit into the main categories.

## Prerequisites

- PostgreSQL running (`./cookbook/scripts/run_pgvector.sh`)
- `OPENAI_API_KEY` set

## Files

| File | Description |
|------|-------------|
| `background_execution.py` | Background execution with polling and cancellation support |

## Running

```bash
.venvs/demo/bin/python cookbook/03_teams/other/background_execution.py
```
