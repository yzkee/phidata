# memory

Examples for team workflows in memory.

## Prerequisites

- Load environment variables (for example, OPENAI_API_KEY) via direnv allow.
- Use .venvs/demo/bin/python to run cookbook examples.
- Some examples require additional services (for example PostgreSQL, LanceDB, or Infinity server) as noted in file docstrings.

## Files

- 01_team_with_memory_manager.py - Demonstrates team with memory manager.
- 02_team_with_agentic_memory.py - Demonstrates team with agentic memory.
- 03_memories_in_context.py - Demonstrates add_memories_to_context.
- learning_machine.py - Demonstrates learning machine.
