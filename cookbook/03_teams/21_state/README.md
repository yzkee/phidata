# state

Examples for team workflows in state.

## Prerequisites

- Load environment variables (for example, OPENAI_API_KEY) via direnv allow.
- Use .venvs/demo/bin/python to run cookbook examples.
- Some examples require additional services (for example PostgreSQL, LanceDB, or Infinity server) as noted in file docstrings.

## Files

- agentic_session_state.py - Demonstrates agentic session state.
- change_state_on_run.py - Demonstrates change state on run.
- nested_shared_state.py - Demonstrates nested shared state.
- overwrite_stored_session_state.py - Demonstrates overwrite stored session state.
- state_sharing.py - Demonstrates state sharing.
