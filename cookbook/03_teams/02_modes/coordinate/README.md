# Coordinate Mode

Coordinate mode examples demonstrate member selection, explicit routing instructions, and synthesis.

## Files

- `01_basic.py` - Basic coordination with specialist members.
- `02_with_tools.py` - Coordination with tool-enabled members.
- `03_structured_output.py` - Structured output examples for coordinated responses.

## Running

```bash
.venvs/demo/bin/python cookbook/03_teams/02_modes/coordinate/01_basic.py
```
