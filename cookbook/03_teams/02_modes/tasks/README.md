# Task Mode

Task mode examples include both baseline and advanced workflows for dependency-aware planning.

## Files

- `01_basic.py` - Basic task decomposition and execution.
- `02_parallel.py` - Parallel task execution example.
- `03_dependencies.py` - Tasks with explicit dependency handling.
- `04_basic_task_mode.py` - Foundational autonomous task-mode flow.
- `05_parallel_tasks.py` - Parallel execution with independent work streams.
- `06_task_mode_with_tools.py` - Task mode using web-search equipped members.
- `07_async_task_mode.py` - Async execution with `arun()`.
- `08_dependency_chain.py` - Complex dependency chains across tasks.
- `09_custom_tools.py` - Task mode with custom Python member tools.
- `10_multi_run_session.py` - Multi-run session persistence patterns.

## Running

```bash
.venvs/demo/bin/python cookbook/03_teams/02_modes/tasks/04_basic_task_mode.py
```
