# dependencies

Examples for team workflows in dependencies.

## Prerequisites

- Load environment variables (for example, OPENAI_API_KEY) via direnv allow.
- Use .venvs/demo/bin/python to run cookbook examples.
- Some examples require additional services (for example PostgreSQL, LanceDB, or Infinity server) as noted in file docstrings.

## Files

- dependencies_in_context.py - Demonstrates dependencies in context.
- dependencies_in_tools.py - Demonstrates dependencies in tools.
- dependencies_to_members.py - Demonstrates dependencies to members.
