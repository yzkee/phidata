# Workflow Observability

Examples for tracing Agno workflows.

## Files

- [`arize_phoenix_via_openinference_workflow.py`](./arize_phoenix_via_openinference_workflow.py): Send workflow traces to Arize Phoenix.
- [`langfuse_via_openinference_workflows.py`](./langfuse_via_openinference_workflows.py): Send workflow traces to Langfuse.

## Run

```bash
.venvs/demo/bin/python cookbook/92_integrations/observability/workflows/<file>.py
```
