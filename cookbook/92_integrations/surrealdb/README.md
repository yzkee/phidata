# SurrealDB Integration

Examples showing SurrealDB as a backend for Agno memory management.

## Files

- [`standalone_memory_surreal.py`](./standalone_memory_surreal.py): Manual add/get/delete/replace memory operations.
- [`memory_creation.py`](./memory_creation.py): Create memories from text and message histories.
- [`custom_memory_instructions.py`](./custom_memory_instructions.py): Capture memories with custom extraction instructions.
- [`memory_search_surreal.py`](./memory_search_surreal.py): Search memories with multiple retrieval methods.
- [`db_tools_control.py`](./db_tools_control.py): Control add/update/delete/clear memory operations.

## Run

```bash
.venvs/demo/bin/python cookbook/92_integrations/surrealdb/<file>.py
```
